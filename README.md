# 🐉 道缘占卜馆 - 天机秘册

## 项目介绍

这是一款融合中国传统文化元素和现代高级交互的**动态占卜网站**。采用古典中国风设计，整合八卦、干支、五行、生肖等传统文化系统，提供精美的视觉效果和沉浸式的占卜体验。

---

## ✨ 高级特效一览

### 1️⃣ **视觉层次**
- **古典中国风配色**：金铜色(#d4a574)、棕色(#8b7355)、深褐色背景
- **八卦旋转背景**：动态旋转的两个八卦圆盘，营造神秘气氛
- **水墨粒子效果**：30个浮动的水墨颗粒，模拟国画氛围
- **边框光晕动画**：容器边框的呼吸式光晕效果

### 2️⃣ **交互特效**
- **八卦盘交互**：
  - 点击时产生8个汉字粒子(卦、象、道、缘、天、地、人、和)
  - 触发笔画特效(4条辐射线)
  - 毛笔笔画渐隐动画

- **鼠标跟随粒子**：
  - 移动鼠标时自动生成✦符号粒子
  - 浮起消散的运动轨迹
  - 按钮点击时产生爆炸效果(12个✨粒子)

- **输入框震动**：
  - 点击"开始占卜"时，三个输入框产生轻微震动

### 3️⃣ **加载动画**
- **三层旋转圆环**：
  - 外圈顺时针旋转(20秒)
  - 中圈逆时针旋转(30秒)
  - 内圈顺时针旋转(16秒)
- **"占卜中"文字**：闪烁的光晕效果

### 4️⃣ **结果展示动画**
- **连锁卡片弹出**：
  - 八卦象征卡片(200ms延迟)
  - 四柱干支卡片(400ms延迟)
  - 五行属性卡片(600ms延迟，伴随粒子)
  - 生肖运势卡片(800ms延迟)
  - 最终预测卡片(1000ms延迟)

- **卡片动画**：
  - 3D翻转进入(rotateY 45deg → 0)
  - 流光闪过效果(从左到右扫过)
  - 不同延迟形成级联效果

- **生肖圆形展示**：
  - 12个生肖图标旋转缩放进入
  - 当前生肖脉冲放大效果
  - 每个图标有50ms错开延迟

### 5️⃣ **五行粒子特效**
- **15个五行字符飘升**：
  - 金(金黄色)、木(绿色)、水(蓝色)、火(橙红色)、土(棕色)
  - 每100ms生成一个
  - 3秒内从屏幕中心飘升到顶部并消散

### 6️⃣ **能量释放效果**
- **20个粒子爆发**：
  - 从容器中心辐射向四周
  - 1.5秒内完成缩放和消散
  - 30ms的间隔创建连贯的爆发感

## 🎯 功能特性

### 核心系统集成
| 系统 | 说明 | 实现 |
|------|------|------|
| **八卦象征** | 8个八卦对应命格分析 | 乾、坤、震、艮、坎、离、兑、风 |
| **天干地支** | 10天干+12地支 | 四柱八字(年月日时) |
| **五行属性** | 5种五行系统 | 金、木、水、火、土 |
| **生肖运势** | 12生肖运势分析 | 个性化预测文本 |
| **预测等级** | 3个预测等级 | 吉象/中象/凶象 |

### 数据计算
- **八字推算**：基于输入的生日自动计算年月日时的干支
- **八卦映射**：根据干支组合推算对应的八卦象征
- **伪随机选择**：根据姓名长度、生日、性别等参数选择预测结果

## 🚀 使用说明

### 基础使用
1. **填写信息**：
   - 输入您的名字(1-20字)
   - 选择您的生日(日期选择器)
   - 选择性别(男/女)

2. **开始占卜**：
   - 点击"🔮 开始占卜"按钮
   - 等待2秒钟的加载动画
   - 观看卡片级联展示

3. **查看结果**：
   - 八卦象征：您的命格卦象
   - 四柱干支：年月日时的天干地支
   - 五行属性：您的五行属性及含义
   - 生肖运势：您的生肖及运势分析
   - 最终预测：吉/中/凶的详细预测

4. **重新开始**：
   - 点击"重新开始"按钮清空所有信息

### 交互技巧
- 移动鼠标在页面上产生粒子轨迹
- 点击八卦盘产生笔画特效
- 点击按钮产生爆炸粒子效果
- 鼠标悬停在生肖上会放大显示

## 📊 结果解释

### 八卦含义
- **乾**：天地之气，刚强坚毅
- **坤**：大地之力，柔顺包容
- **震**：雷霆之势，震撼人心
- **艮**：高山之壮，坚实不移
- **坎**：流水之润，循环往复
- **离**：烈火之明，光彩照人
- **兑**：清水之喜，悦人心扉
- **风**：风行之力，无处不至

### 五行特性
- **金**：刚强内敛，处事有度
- **木**：欣欣向荣，生机勃勃
- **水**：聪慧灵动，知变适变
- **火**：热情似火，才华横溢
- **土**：厚重稳实，诚实守信

### 预测等级
- **吉象**：运势上升，机遇众多
- **中象**：平稳前行，需要努力
- **凶象**：挑战重重，需要谨慎
